# Lecturas base

Biblioteca del curso organizada por autor o coautoría.

## Convenciones

- Los nombres de archivo usan `snake_case`.
- El PDF conserva la fuente visual original.
- El sufijo `_ocr.md` identifica una extracción automática consultable, no una
  edición académica corregida.
- Las fichas, notas y reseñas elaboradas para el curso se distinguen del OCR.
- Cuando el OCR era vacío o no contenía la obra, se eliminó y se conservó el
  PDF con la indicación `OCR pendiente`.

## Ejes del curso

- **Ontología y experiencia del espacio:** Agamben, Álvarez Falcón, Flores,
  Gutiérrez Hernández, Heidegger, Hidalgo Hermosilla y Quesada.
- **Poder, espacio público y gobierno urbano:** Carrión, Castells, Foucault,
  Jacobs, Martínez Posada, Ramírez Kuri, Stavrides y Zuboff.
- **Modernidad y formas urbanas:** Berman, Calvino, Chueca Goitia, Cruz,
  Koolhaas, Ruiz del Olmo, Sassen y Sennett.
- **Técnica e IA urbana:** Blasco Ejarque y Tirado, Cugurullo y colaboradores,
  Diéguez, Echeverría y Tabares, Ortega y Gasset, Terrones Rodríguez y Yuk Hui.
