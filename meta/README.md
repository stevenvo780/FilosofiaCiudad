# meta/ — Material de contexto y configuración del proyecto

Esta carpeta reúne los archivos de contexto y configuración que orientan el trabajo de agentes y asistentes en el proyecto **Filosofía de la ciudad: ontología, poder y política**. No contiene notas de clase ni materiales académicos del curso; su función es exclusivamente operativa y meta-editorial.

## Contenido de esta carpeta

| Archivo | Descripción |
|---|---|
| `asesor_curso_filosofia_ciudad.prompt.md` | Prompt base del asistente académico del curso. Define el rol, el nivel de rigor esperado, el marco conceptual y el modo de trabajo. |
| `st-guide.md` | Guía del lenguaje de lógica formal ST (`st`). Incluye sintaxis, operadores, perfiles de lógica disponibles y ejemplos completos. |

## Archivos de contexto en el resto del repositorio

Los siguientes archivos complementan o extienden el contexto de esta carpeta y deben mantenerse alineados entre sí:

- [AGENTS.md](../AGENTS.md) — Instrucciones para Codex: propósito del repositorio, rol esperado, convenciones de edición y entorno de trabajo.
- [.github/copilot-instructions.md](../.github/copilot-instructions.md) — Instrucciones equivalentes para GitHub Copilot.
- [asesor_curso_filosofia_ciudad.prompt.md](asesor_curso_filosofia_ciudad.prompt.md) — Prompt del asistente académico (este archivo).
- [st-guide.md](st-guide.md) — Guía del intérprete ST de lógica formal.
