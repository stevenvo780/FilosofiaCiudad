# Filosofia de la Ciudad

Archivo academico y tecnico del curso **Filosofia de la ciudad: ontologia,
poder y politica**.

## Sistema de archivos

| Carpeta | Contenido |
|---|---|
| `01_Clases/` | Apuntes, borradores y recursos organizados por clase |
| `02_Lecturas_Base/` | Fuentes, fichas y notas agrupadas por autor |
| `03_Trabajos/` | Ponencias y otros entregables |
| `04_Recursos_Tecnicos/` | Scripts, prompts y dependencias |
| `repos/` | Repositorios locales independientes, no sincronizados |

## Criterio de organizacion

- Los nombres nuevos usan `snake_case` y evitan titulos genericos.
- Cada seccion tiene un `README.md` como indice.
- Las fuentes y sus versiones OCR se conservan juntas cuando cumplen funciones
  distintas.
- Los borradores se mantienen separados de las notas listas para estudio.
- Los archivos generados permanecen junto al material que documentan.

## Entradas principales

- `01_Clases/README.md`: recorrido completo de las clases.
- `02_Lecturas_Base/README.md`: biblioteca por autor.
- `03_Trabajos/README.md`: entregables y estado.
- `04_Recursos_Tecnicos/README.md`: ejecucion de scripts y prompts.
